## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)
* [TypeScript](https://www.typescriptlang.org/)
* [Jest](https://jestjs.io/)

## Istallation

```sh
npm i
```
or

```sh
yarn
```

## Running test

```sh
node test
```
or

```sh
yarn test
```
